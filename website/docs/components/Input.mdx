## Props

:::note
Includes all [React Native TextInput](https://reactnative.dev/docs/textinput#props), [View](https://reactnative.dev/docs/view#props) props.
:::

<div class='table-responsive'>

| Name                      | Type            | Default | Description                      |
| ------------------------- | --------------- | ------- | -------------------------------- |
| `InputComponent`          | React Component |         | 可替换默认提供的 TextInput       |
| `containerStyle`          | View Style      |         | 最外层 View Style                |
| `disabled`                | boolean         |         | 禁止输入                         |
| `disabledInputStyle`      | Text Style      |         | 禁止输入后的样式                 |
| `errorMessage`            | string          |         | 错误提示信息                     |
| `errorProps`              | object          |         | 错误提示信息 Props               |
| `errorStyle`              | Text Style      |         | 错误提示信息样式                 |
| `inputContainerStyle`     | View Style      |         | 暂无用处                         |
| `inputStyle`              | Text Style      |         | /\*\* TextInput 输入框样式       |
| `label`                   | ReactNode       |         | 组件 label 暂无用处              |
| `labelProps`              | object          |         | label Props                      |
| `labelStyle`              | Text Style      |         | label 样式                       |
| `leftIcon`                | IconNode        |         | 显示左侧 Icon 图标               |
| `leftIconContainerStyle`  | View Style      |         | 包裹左侧 Icon 图标外层 View 样式 |
| `renderErrorMessage`      | boolean         |         | 是否显示错误信息                 |
| `rightIcon`               | IconNode        |         | 显示右侧 Icon 图标               |
| `rightIconContainerStyle` | View Style      |         | 包裹右侧 Icon 图标外层 View 样式 |

</div>
