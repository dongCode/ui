Buttons are touchable elements used to interact with the screen and to perform and operation. They may display text, icons, or both. Buttons can be styled with several props to look a specific way.

### Variants

There are solid button, outline button and clear button types

```tsx live
<Stack row align="center" spacing={4}>
  <Button title="Solid" />
  <Button title="Outline" type="outline" />
  <Button title="Clear" type="clear" />
</Stack>
```

### Size

Button can be small medium or large

```tsx live
<Stack row align="center" spacing={4}>
  <Button size="sm">Small</Button>
  <Button size="md">Medium</Button>
  <Button size="lg">Large</Button>
</Stack>
```

### Colors

```tsx live
<Stack row align="center" spacing={4}>
  <Button>Primary</Button>
  <Button color="secondary">Secondary</Button>
  <Button color="warning">Warning</Button>
  <Button color="error">Error</Button>
</Stack>
```

### Disabled

```tsx live
<Stack row align="center" spacing={4}>
  <Button title="Solid" disabled />
  <Button title="Outline" type="outline" disabled />
  <Button title="Clear" type="clear" disabled />
</Stack>
```

### Linear Gradient

```tsx live
<Button
  ViewComponent={LinearGradient} // Don't forget this!
  linearGradientProps={{
    colors: ["#FF9800", "#F44336"],
    start: { x: 0, y: 0.5 },
    end: { x: 1, y: 0.5 },
  }}
>
  Linear Gradient
</Button>
```

### Icon Button

Can contain an Icon by setting the icon prop or placing an Icon component within the Button.

```tsx live
<Button radius={"sm"} type="solid">
  Save
  <Icon name="save" color="white" />
</Button>
```

### Loading spinner

```tsx live
<Button title="Solid" type="solid" loading />
```

## Props

:::note
Includes all [TouchableOpacityProps](https://reactnative.dev/docs/touchableopacity#props) props.
:::

<div class='table-responsive'>

| Name                  | Type                                                                      | Default | Description                                                            |
| --------------------- | ------------------------------------------------------------------------- | ------- | ---------------------------------------------------------------------- |
| `TouchableComponent`  | typeof Component                                                          |         | 用户交互组件 默认 TouchableOpacity，会根据 ios android 平台调整        |
| `ViewComponent`       | React Component                                                           |         | 默认值 View 与 linearGradientProps 配合使用                            |
| `buttonStyle`         | View Style                                                                |         | 自定义按钮样式                                                         |
| `color`               | `string` \| `primary` \| `secondary` \| `success` \| `warning` \| `error` |         | 颜色                                                                   |
| `containerStyle`      | View Style                                                                |         | 最外层 View 样式                                                       |
| `disabled`            | boolean                                                                   | `false` | 禁用                                                                   |
| `disabledStyle`       | View Style                                                                |         | 禁用样式                                                               |
| `disabledTitleStyle`  | Text Style                                                                |         | 禁用文字样式                                                           |
| `icon`                | IconNode                                                                  |         | 图标                                                                   |
| `iconContainerStyle`  | View Style                                                                |         | 包裹图标的 View 样式                                                   |
| `iconPosition`        | `left` \| `right` \| `top` \| `bottom`                                    | `left`  | 定义图标显示位置                                                       |
| `iconRight`           | boolean                                                                   | `false` | 按钮显示在右边，需要配合 icon 属性使用                                 |
| `linearGradientProps` | object                                                                    |         | 渐进样式 详情 [用法](#linear-gradient).                                |
| `loading`             | boolean                                                                   | `false` | 加载提示                                                               |
| `loadingProps`        | ActivityIndicatorProps                                                    |         | 加载 View 属性                                                         |
| `loadingStyle`        | View Style                                                                |         | 加载 View 样式                                                         |
| `radius`              | `number` \| `sm` \| `md` \| `lg`                                          | `xs`    | 按钮 radius                                                            |
| `raised`              | boolean                                                                   | `false` | Add raised button styling (optional). Has no effect if `type="clear"`. |
| `size`                | `sm` \| `md` \| `lg`                                                      | `md`    | 大小                                                                   |
| `title`               | `string` \| `ReactElement<{}, string` \| `JSXElementConstructor<any>>`    |         | 文字                                                                   |
| `titleProps`          | TextProps                                                                 |         | 包裹文字 Text 属性                                                     |
| `titleStyle`          | Text Style                                                                |         | 包裹文字组件 Text 样式                                                 |
| `type`                | `solid` \| `clear` \| `outline`                                           | `solid` | 显示按钮类型 solid 默认值 clear 文字按钮 outline 仅显示边框            |
| `uppercase`           | boolean                                                                   | `false` | 文字大写                                                               |

</div>
