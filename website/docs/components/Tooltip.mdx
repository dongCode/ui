### Example

```tsx live
function RNETooltip() {
  const [open, setOpen] = React.useState(false);
  return (
    <Stack row align="center">
      <Tooltip
        visible={open}
        onOpen={() => setOpen(true)}
        onClose={() => setOpen(false)}
        popover={<Text style={{ color: "#fff" }}>Tooltip text</Text>}
      >
        Click me
      </Tooltip>
    </Stack>
  );
}
```

## Props

<div class='table-responsive'>

| Name                       | Type                                                       | Default       | Description                                            |
| -------------------------- | ---------------------------------------------------------- | ------------- | ------------------------------------------------------ |
| `ModalComponent`           | typeof Component                                           |               | 覆盖 Modal 弹窗组件                                    |
| `animationType`            | `none` \| `fade`                                           | `fade`        | 动画样式                                               |
| `backgroundColor`          | ColorValue                                                 | `#617080`     | 设置背景颜色                                           |
| `closeOnlyOnBackdropPress` | boolean                                                    | `false`       | 点击遮罩层是否关闭                                     |
| `containerStyle`           | View Style                                                 | `{}`          | 最外层 View 样式                                       |
| `height`                   | number                                                     | `40`          | 根据渲染内容，可自定义高度                             |
| `highlightColor`           | ColorValue                                                 | `transparent` | 高亮显示内容颜色                                       |
| `onClose`                  | Function                                                   | `Function`    | 关闭时的回调函数                                       |
| `onOpen`                   | Function                                                   | `Function`    | 打开时的回调函数                                       |
| `overlayColor`             | ColorValue                                                 | `#fafafa14`   | 遮罩层颜色                                             |
| `pointerColor`             | ColorValue                                                 | `#617080`     | 箭头颜色默认值是 [`backgroundColor`](#backgroundcolor) |
| `pointerStyle`             | View Style                                                 |               | 箭头样式                                               |
| `popover`                  | `ReactElement<{}, string` \| `JSXElementConstructor<any>>` |               | 自定义弹出组件                                         |
| `skipAndroidStatusBar`     | boolean                                                    | `false`       | 如果嵌套在原生 Modal 中，需要传递 true                 |
| `toggleAction`             | string                                                     | `onPress`     | 自定义响应事件类型                                     |
| `toggleOnPress`            | boolean                                                    | `true`        | 是否支持自身控制显示隐藏                               |
| `visible`                  | boolean                                                    | `false`       | 是否显示                                               |
| `width`                    | number                                                     | `150`         | 根据渲染内容，可自定义宽度                             |
| `withOverlay`              | boolean                                                    | `true`        | 是否显示遮罩层                                         |
| `withPointer`              | boolean                                                    | `true`        | 是否显示小箭头                                         |

</div>
