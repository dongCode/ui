### react-native-vector-icons 问题


[Expo](https://expo.io) or
[create-react-native-app](https://github.com/react-community/create-react-native-app)
创建的项目包含 **react-native-vector-icons** out of the box, hence this step can be skipped.

如果项目是使用
`react-native init`创建的 , 那么你需要引入 `react-native-vector-icons`.



### react-native-safe-area-context 问题

需要引入  **react-native-safe-area-context** 包，可以采用以下命令引入


<Tabs
defaultValue="npm"
groupId='pkg'
values={[
{ label: 'NPM', value: 'npm', },
{ label: 'Yarn', value: 'yarn', },
]
}>
<TabItem value="npm">

```bash
npm install react-native-safe-area-context
```

</TabItem>
<TabItem value="yarn">

```bash
yarn add react-native-safe-area-context
```

</TabItem>
</Tabs>


需要使用 `SafeAreaProvider` 包裹根组件，例如：
```js
import { SafeAreaProvider } from 'react-native-safe-area-context';

function App() {
  return <SafeAreaProvider>...</SafeAreaProvider>;
}
```
